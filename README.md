# Temperature Converter (Python Project)

1. Overview
This is a simple Python project made for beginners to convert temperature values between Celsius, Fahrenheit, and Kelvin. The program takes user input, applies the correct conversion formula, and displays the result.  
It is an easy and useful project for understanding basic Python concepts like input/output, conditional statements, and arithmetic operations.

2. Features
- Convert Celsius to Fahrenheit  
- Convert Fahrenheit to Celsius  
- Convert Celsius to Kelvin  
- Convert Kelvin to Celsius  
- Simple menu-driven interface  
- Beginner-friendly and easy to understand
  
3. Concepts Used
- input() for taking user input  
- float() for numeric values  
- if-elif-else for decision making  
- Basic mathematical formulas  

4. Conversion Formulas
   Celsius → Fahrenheit: 
   F = (C × 9/5) + 32

5. Fahrenheit → Celsius: 
    C = (F – 32) × 5/9

6. Celsius → Kelvin: 
    K = C + 273.15

7. Kelvin → Celsius:
   C = K – 273.15


8. How to Run the Program
1. Install Python on your system.  
2. Create a .py file and copy the program code inside it.  
3. Open terminal or command prompt.  
4. Run the file using the command:  
python filename.py
